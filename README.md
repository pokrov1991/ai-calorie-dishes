# Мультимодальная модель для предсказания калорийности блюд

Мльтимодальная модель, которая предсказывает калорийность блюда по его изображению.

## Структура проекта

```bash
ai-calorie-dishes/
│
├── data/
│   ├── dish.csv                # основной CSV с блюдами и сплитами
│   ├── ingredients.csv         # ингредиенты
│   └── images/                 # изображения блюд (dish_id/rgb.png)
│
├── models/
│   ├── dish_calorie_model.pt   # обученная модель (чекпоинт)
│   └── ingredient_vocab.json   # словарь ингредиентов
│
├── scripts/
│   ├── config.py               # конфигурация проекта
│   ├── dataset.py              # работа с датасетом, Dataset + DataLoader
│   ├── utils.py                # модель, обучение, метрики
│   └── inference.py            # инференс и оценка модели
│
├── notebook.ipynb              # ноутбук со всеми этапами проекта
├── requirements.txt            # зависимости проекта
└── README.md                   # описание проекта
